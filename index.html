OSX Shortcuts

Basic key bindings for OSX

This is a very cut down version of the keystrokes and tips that I find useful for OSX.

Finder

Action Shortcut
→ In List view, expand selected folder
← In List view, collapse selected folder
Enter Rename the selected file/folder
⌘↓ Go into selected folder or open the selected file
⌘↑ Go to parent folder
⌘[ Go Back
⌘] Go Forward
⌘C, then ⌘V Copy and Paste files
⌘C hen ⌘⌥⇧V Copy and Paste files Exactly (Keeps original ownership & permissions. Requires Admin username & password.)
⌘C, then ⌘-⌥–V Cut and Paste files (Move files)
⌘ Drag file to disk Move a file instead of copying (Copies to the destination and removes it from the original disk.)
⌘⌥Delete Immediately Delete a File (does not put file in Trash)
Esc Cancel a drag-n-drop action while in the midst of dragging
⌘Z Undo the last action (such as rename file, copy file, etc.)
Pinch on a trackpad or hold ⌥ while scrolling. For images, hold ⌥ to zoom 100%. Zoom In/Out in Quick Look (Images & PDFs)
⌘F Find Files

Sleep, log out, and shut down

Action Shortcut
Tap power button Turn on your Mac or wake your Mac from sleep
Hold for 5 seconds to force your Mac to turn off Turn off Mac
⌃ Power button. Restart Mac
⌘⌃ Power button Force your Mac to restart
⇧⌃ Power button Put your displays to sleep
⌘⇧⌥Q Log out of your OS X user account immediately, without being asked to confirm

Document Shortcuts (Doesn't work in all apps)

Action Shortcut
⌘→ Go to end of line
⌘← Go to beginning of line
⌘↓ Go to end of all the text
⌘↑ Go to beginning of all the text
⌥→ Go to end of current or next word
⌥← Go to beginning of current or previous word
Add ⇧ to the above keystrokes to make a selection to that point.
⌥ Drag Mouse Select non-linear areas
⌥-Delete Delete entire word to the left
⌘⌥⇧V Paste and Match Style (so it looks the same as the text you are pasting into)
Mouse over a word and hit ⌘-Ctrl-D (Or on a trackpad use 3 fingers to tap on a word) Look up word in dictionary

Screenshots

Action Shortcut
⌘⇧3 Take picture of the entire screen
⌘⇧4 and Drag over an area. While dragging:-Hold Spacebar to move selected area.-Hold ⇧ to change size in one direction only (horizontal or vertical)-Hold ⌥ for center-based resizing. Take picture of a selected area
⌘⇧4, then press Spacebar, then Click on the window/object Take picture of a specific window/object
Hold ⌃ with any of the above keystrokes to copy the screenshot to the clipboard instead of making a file.

Other

Action Shortcut
⌥Mouse Click on the wireless icon Show extended wireless information

VS Code Shortcuts

Basic key bindings for VS Code.

This is a very cut down version of the keystrokes and tips that I find useful for VSCode.

For full list see: VS Code Website

In addition some of the tips are taken from the course Visual Studio Code by John Papa

Only OSX keystrokes are shown, but as Visual Code is a multiplatform browser, they are generally the same once the appropriate OS equivalent key is hit.

Windows OSX
Windows-Key ⌘/Command
Ctrl ⌃/Control
Alt ⌥/Option
Shift ⇧/Shift
↓ ↓
↑ ↑
← ←
→ →

General

Keystroke Behaviour
⌘P Go to file
⌘PP Go to previous file
⌘⇧P Open main command Palette (useful for finding out more vs features)
⌃⌘F Full Screen Mode
⌘B Toggle Sidebar

General Editing

Keystroke Behaviour
⌘X|Split Editor Pane (up to three times)
⌘2 Focus on editor pane 2
⌘1 Focus on editor pane 1
⌘3 Focus on editor pane 3
⌥⌘← Focus into Next Editor on the Right
⌥⌘→ Focus into Next Editor on the Left
⌘⇧] Jump to matching bracket
⌘W Close current editor pane
⌘] Outdent Line
⌘[ Indent line
⇧⌘V Toggle Preview Mode

Multi cursor mode

This group of commands is quite powerful but takes a bit of playing around with to get your head around.

Keystroke Behaviour
⌘D Add current item to Find Match and move to next
⌘K ⌘D Move Last Selection To Next Find Match
⇧⌘L Add all identified items to the Find Match.
⌥⌘↓ Start a multi cursor on the line below. This also works for above.
⌥ Mouse Click Place a multi cursor at position of mouse click

Rich Language editing

Keystroke Behaviour
⇧⌥F Format Text
⌃Space Trigger Suggestion (Schema based)

Eratta

VSCode comes with a number of built in tools to aid the development experience:

Emmet

Emmet, "the essential tool for web developers" is built into VS Code by default.

By typing the following on an empty html page and then hitting tab, you will get a simple to do list.

Snippets

User snippets can be added through the Code>Preferences>User Snippets menu. They are sensitive to the selected language.
